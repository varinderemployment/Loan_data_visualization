 Loan Data Visualization | Power BI | Excel

 Project Overview
 
This project focuses on analyzing and visualizing *loan data* using *Microsoft Power BI* to understand loan distribution, customer profiles, and risk patterns.  
The dashboard helps in identifying trends related to loan amount, approval status, income level, and credit behavior.



 Objectives
 
- Analyze loan approval patterns  
- Understand customer demographics  
- Identify risk factors affecting loan approval 
- Present insights through interactive dashboards  



 Tools & Technologies
 
- Power BI
- Excel (Data Source)
- Data Cleaning & Transformation
- DAX (Basic Measures)


 Key Visualizations
 
- Loan Approval vs Rejection  
- Loan Amount Distribution  
- Applicant Income Analysis  
- Credit History Impact  
- Gender & Marital Status based insights
